Syncing Zotero attachments via Box.com

By Adam M. Wilson August 20, 2014 Comment

I use the wonderful Zotero reference manager but have been frustrated by it’s small storage for attachments (which I want to sync to read on multiple devices).

  1. Zotero Storage is quite expensive ($60/year for 6GB). Given that I’m already paying $120/year to Google Drive for 1TB, I’d rather use that storage directly.
  2. They only support WebDaav backups which seem to be getting rarer and rarer. Neither Dropbox or Google Drive currently support the WebDaav protocols.

However box.com provides a full 10GB for free and provides webDaav access. Awesome.
